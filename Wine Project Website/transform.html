A Quantitative Comparison Analysis of Spanish Wine Scores vs Weather Data.

Is there a correlation between temperature, precipitation and wine scores?

Wine Review data transformation:

  1. Pulled JSON file into Jupyter Notebook.
  2. Read the JSON file into a Pandas DataFrame.
  3. Dropped unneeded columns.
  4. Dropped all rows where country != 'Spain'.
  5. Used a 'For' loop to search for year strings in Title.
  6. Appended the years to a list in the order they were found in.
  7. Remove wines where no year was present in Title.
  8. Remove wines that weren't harvested between 2000-2012.
  9. Append the list of years.
  10. Push data to MySQL.
  11. Confirm the tables.
  12. Use MatPlotLib to create scatter plot.

Weather data transformation:

  1. Pulled in three .csv files with 5 years worth of weather data each.
  2. Removed null values.
  3. Paired down to just the columns needed.
  4. Transferred the rain data from a string to float.
  5. Found that we had non numerical values in the rain column.
  6. Removed the non numerical values.
  7. Successfully transferred rain data to float.
  8. Averaged data grouped by year.
  9. Reset index so that year could be a column in sql.
  10. Exported to mysql.
